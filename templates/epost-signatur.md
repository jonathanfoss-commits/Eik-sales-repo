# E-postsignatur

Standard signatur for Jonathans utgående e-post. Hold den enkel, profesjonell og konsistent på tvers
av henvendelser, oppfølginger og tilbud.

## Standard (norsk)
```
Beste hilsen,
Jonathan Foss
Sales Manager · Eik & Friends
{{telefon}} · {{epost}}
{{nettside}}
```

## Kort (for oppfølginger i samme tråd)
```
— Jonathan, Eik & Friends
```

## Engelsk variant (internasjonale mottakere)
```
Best regards,
Jonathan Foss
Sales Manager · Eik & Friends
{{telefon}} · {{epost}}
{{nettside}}
```

## Retningslinjer
- Bruk den **standard** signaturen på første henvendelse og i tilbud.
- Bruk den **korte** i pågående tråder for å holde meldingene lette.
- Match språket i signaturen til språket i e-posten.
- Ikke legg til sitater, bilder eller ansvarsfraskrivelser med mindre det er nødvendig — kortere er
  mer troverdig.
