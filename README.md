# Lærling — AI-medarbeideren som bygger seg selv rundt bedriften din

> Abonnementstjeneste fra Eik & Friends (arbeidstittel: **Lærling**)

**Én setning:** Lærling er en AI-agent som undersøker bedriften din, lærer hvordan du jobber, og deretter *utvikler og drifter* skreddersydde AI-verktøy og automasjoner for deg — helt uten at du trenger å kunne noe om AI.

## Hvem er det for?

AI-nybegynnere i små og mellomstore bedrifter: håndverkere, klinikker, butikker, regnskapsførere, eiendomsmeglere, byråer — folk som vet at AI kan hjelpe dem, men som ikke aner hvor de skal begynne.

## Hvordan det fungerer

1. **Undersøker** — Lærling leser nettsiden din, offentlige registre, og (med samtykke) e-post, kalender og fagsystemer. Den intervjuer deg på vanlig norsk.
2. **Lærer** — Den bygger en levende bedriftsprofil og finner de mest tidkrevende, repetitive oppgavene dine.
3. **Utvikler** — Den foreslår og bygger konkrete AI-verktøy for *din* bransje: tilbudsgenerator, booking-svar, journalnotater, purringer, innholdsproduksjon.
4. **Opererer** — Verktøyene kjører automatisk. Lærling overvåker, forbedrer og rapporterer hver uke. Du godkjenner — den gjør jobben.
