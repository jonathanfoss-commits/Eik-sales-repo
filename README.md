# Flyttet

SAGA (assistent + styre + selskapsfabrikk) er skilt ut som eget, personlig
sideprosjekt og bor nå i **https://github.com/jonathanfoss-commits/saga**
(app: https://jonathanfoss-commits.github.io/saga/).

Dette repoet inneholder ikke lenger SAGA-kode. Gamle SAGA-adresser under
dette repoets Pages redirecter automatisk til den nye appen; nettleserdata
og nøkler er uberørt (samme origin). Redirect-stubbene kan slettes når
bokmerker er oppdatert.
